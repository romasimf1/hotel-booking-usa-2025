1. GETTING API KEYS
========================

A) EmailJS Setup:
   1. Go to https://www.emailjs.com/
   2. Register and sign in to your account
   3. Create a new service (Email Services) - use Gmail or Outlook
   4. Create an email template (Email Templates) with this content:

   Subject: New Hotel Booking Request

   Body:
   Hello!

   New booking request received:

   Guest: {{guest_name}}
   Email: {{guest_email}}
   Phone: {{guest_phone}}
   Hotel: {{hotel_name}}
   Check-in: {{check_in}}
   Check-out: {{check_out}}
   Guests: {{guests}}
   Request Date: {{booking_date}}

   Please contact the guest to confirm the booking.

   Best regards,
   Hotel Booking System USA 2025

   5. Copy Service ID, Template ID, and Public Key

B) ElevenLabs Setup:
   1. Go to https://elevenlabs.io/
   2. Register and sign in to your account
   3. Go to Conversational AI
   4. Create a new agent (Agent)
   5. Configure voice and agent behavior
   6. Copy Agent ID from URL or settings

2. PROJECT CONFIGURATION
=====================

Open script.js file and replace the following lines:

// Replace with your EmailJS Public Key
emailjs.init("YOUR_EMAILJS_PUBLIC_KEY");

// Replace with your EmailJS Service ID and Template ID
await emailjs.send(
    'YOUR_EMAILJS_SERVICE_ID',
    'YOUR_EMAILJS_TEMPLATE_ID',
    bookingData
);

// Replace with your ElevenLabs Agent ID
convaiWidget.setAttribute('agent-id', 'YOUR_ELEVENLABS_AGENT_ID');

3. RUNNING THE PROJECT
==================

A) Local run:
   python -m http.server 8000
   Open http://localhost:8000 in browser

B) Deploy to Cloudflare:
   1. Install Wrangler: npm install -g wrangler
   2. Authenticate: wrangler auth login
   3. Deploy: wrangler pages deploy . --name hotel-booking-usa-2025

4. ADDITIONAL CONFIGURATIONS
===========================

- For production, configure domain in Cloudflare
- Add analytics via Google Analytics or Cloudflare Web Analytics
